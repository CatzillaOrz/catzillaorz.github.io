      awk

      • 基本参数和语法

        • NR 行号, NF 列数量
        • $1 第1列, $2, $3…
        • -F fs fs分隔符，字符串或正则
        • 语法: awk ‘BEGIN{ commands } pattern{ commands } END{ commands }’, 流程如下:
          • 执行begin
          • 对输入每一行执行 pattern{ commands }, pattern 可以是 正则/reg exp/, 关系运算等
          • 处理完毕, 执行 end

      • 其他高级语法: for, while 等, 各种函数等, 本身awk是一个强大的语言, 可以掌握一些基本的用法.
